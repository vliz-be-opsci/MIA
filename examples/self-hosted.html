MIA Self-Hosted Example

⚠️ Self-Hosting Setup Required

This example requires you to build MIA locally and serve this file via HTTP server. See the development setup instructions in the README.

Setup Instructions

  1. Clone the MIA repository: git clone https://github.com/vliz-be-opsci/MIA.git
  2. Install dependencies: npm install
  3. Build the project: npm run build
  4. Start HTTP server: http-server
  5. Navigate to http://localhost:8080/examples/self-hosted.html

Advantages of Self-Hosting

  • Full control over configuration and customization
  • No external dependencies for the core MIA functionality
  • Ability to modify templates and styling
  • Custom proxy settings for internal networks
  • Enhanced privacy and security for sensitive applications
