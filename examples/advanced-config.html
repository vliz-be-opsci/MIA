Configuration Details

Current Settings:

  • data-deref-config: Default configuration file with all supported data types
  • data-proxy: CORS proxy for fetching RDF data
  • data-self-reference: Process links that reference the current domain
  • data-extra-properties: Include additional decorative elements
  • data-default-template: Fallback template for unknown data types

Testing Instructions:
